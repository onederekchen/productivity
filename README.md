# productivity

### Requirements

- jupyterlab 2.2.8+
- matplotlib 3.3.2+
- numpy 1.19.2+
- pandas 1.1.3+
- pickleshare 0.7.5+
- scikit-learn 0.23.2+
- seaborn 0.11.0+

### Installation

To generate a model from the latest source, clone the repository and run the classifier script.
```
git clone https://github.com/onederekchen/productivity
cd productivity/model
python classify_data.py
```
